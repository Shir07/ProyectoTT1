# Tienda de Mochi Donuts  

Este proyecto ha sido desarrollado como primera entrega para de Talento Tech. 
La tienda tiene como objetivo ofrecer a los clientes una experiencia sencilla y atractiva para realizar encargos de **Mochi Donuts** a través de WhatsApp.  

## Objetivo  
La tienda proporciona una interfaz visualmente agradable y fácil de usar, permitiendo a los clientes explorar los diferentes sabores de Mochi Donuts y realizar sus pedidos de manera rápida.

## Créditos de Imágenes  
Las imágenes de los Mochi Donuts utilizadas en este proyecto pertenecen a **Deepthi Clicks**.  

## Tecnologías Utilizadas  
- **HTML & CSS** → Para la estructura y el diseño responsivo.  
- **Flexbox** → Para la organización eficiente del diseño.  
- **Google Icons** → Para mejorar la interfaz visual.  

## Encargos vía WhatsApp  
Los clientes podrán seleccionar sus Mochi Donuts favoritos y realizar su pedido directamente a través de **WhatsApp**, garantizando una experiencia de compra rápida y sin complicaciones.
